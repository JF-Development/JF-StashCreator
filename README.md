JF-Development – Stash Creator

A plug-and-play FiveM stash creation and management script that allows admins to create, edit, move, and delete stashes in-game with no long configuration.

Features

Create stashes in-game

Manage existing stashes easily

Move stashes to new locations

Edit or delete stashes at any time

No config editing required

Lightweight and performance-friendly

Built for QBCore

Commands
Command	Description
/stashcreator	Create a stash at your current location
/managestashes	Edit, move, change, or delete existing stashes
Requirements

FiveM Server

group.admin permission

Installation

Download the script

Place the folder into your resources directory

Run the included SQL file in your database

Ensure the resource in your server.cfg:

ensure JF-StashCreator


Make sure your user has the group.admin permission

That’s it no long configuring required.

Usage

Stand where you want the stash

Use /stashcreator to create it

Use /managestashes to manage existing stashes

All stashes are saved automatically.
